6. Click "Purchasing" to view purchase orders or create a new one
7. Click "Forecasting" to see demand predictions per SKU
8. Click "Reports" to view inventory valuation and export CSV
9. The app works without any external API keys for core features
10. Email/Slack/SMS alerts require optional API keys in environment variables
